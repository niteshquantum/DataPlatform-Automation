Reference implementation behavior:

1. Jenkins setup pipeline invokes the database-specific setup scripts.
2. Setup pipeline validates runtimes and tools.
3. Setup pipeline deploys/starts the database service.
4. Setup pipeline creates the database.
5. Setup pipeline runs database table Liquibase changesets.
6. Load pipeline validates runtime dependencies.
7. Load pipeline starts or validates the database service.
8. Load pipeline downloads or verifies the dataset.
9. Load pipeline detects schema and updates Liquibase table master XML.
10. Load pipeline runs Liquibase update.
11. Load pipeline loads data through the shared data loader.
12. Load pipeline validates data and row counts.
13. Object deployment regenerates master_objects.xml from object XML files.
14. Object deployment delegates Liquibase execution to the existing
    database-specific run_liquibase script.
15. Object validation checks the database catalog for declared views,
    functions, stored procedures, and triggers.
